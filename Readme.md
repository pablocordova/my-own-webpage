# MY WEB PAGE 

My source code of my own webpage

## Technologies
* Nodejs 7.2.1
* Express 4.15.2
* Bootstap 4
* JQuery 3.1.1

## To Run

1. Clone or download the repository.
2. Via console go to the file
3. Write the command:
``` node index.js ```
4. In browser write: 
    ``` 
    localhost:5000
    ```
